# Architecture Patterns with Python - Examples

This repository contains some examples for the book [Architecture Patterns with Python](https://www.cosmicpython.com/).

## Getting started

### Install the dependencies

```bash
pipenv install
```

### Activate the virtual environment

```bash
pipenv shell
```

### Run the server

```bash
uvicorn main:app --reload 
```

### Open the docs

```bash
open http://localhost:8000/docs
```
